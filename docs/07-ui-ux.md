## 1. Objetivo

La interfaz debe transmitir software empresarial moderno, limpio y profesional. Debe servir a gerencia, administrativos, comerciales, jefes de equipo y tecnicos.

## 2. Inspiracion

- Linear.
- Notion.
- Monday.
- Atlassian.
- ClickUp.

## 3. Principios

- Mucho espacio visual.
- Paneles antes que ventanas emergentes.
- Iconografia clara.
- Animaciones sutiles.
- Tema claro y oscuro.
- Responsive.
- Accesibilidad.
- Estado de sincronizacion siempre visible en movil.

## 4. Panel web oficina

Debe incluir vistas para:

- Dashboard.
- Clientes.
- Proveedores.
- Puertas.
- Avisos.
- Presupuestos.
- Partes.
- Validacion de oficina.
- Facturacion.
- Documentos.
- Configuracion.

## 5. App movil tecnico

Prioridades:

- `Cargar trabajos` visible.
- Lista de trabajos cargados.
- Estado offline/online.
- Cambios pendientes.
- `Enviar trabajo` visible.
- Captura de fotos sencilla.
- Firma en pantalla.
- Materiales usados.
- Horas y desplazamiento.

## 6. Web publica

Debe ser clara y orientada a conversion:

- Servicios.
- Solicitud de presupuesto.
- Aviso o reparacion.
- Fotos opcionales.
- Consentimiento de proteccion de datos.
- Mensajes de confirmacion.

## 7. Criterios de aceptacion

- El usuario identifica facilmente su modulo.
- La oficina puede validar partes sin friccion.
- El tecnico entiende que datos estan pendientes de envio.
- La web publica genera solicitudes claras para oficina.
