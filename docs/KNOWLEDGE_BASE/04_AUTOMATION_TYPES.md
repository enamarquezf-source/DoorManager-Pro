## 1. Proposito

Definir tipos de automatizacion relevantes para inspeccion, seguridad, operatividad y compatibilidad.

## 2. Tipos iniciales

- Manual.
- Semiautomatico.
- Automatico.
- Hombre presente.
- Automatico subida / hombre presente bajada.

## 3. Impacto en seguridad

El tipo de automatizacion condiciona sistemas de seguridad, puntos de inspeccion, riesgo y uso permitido.
