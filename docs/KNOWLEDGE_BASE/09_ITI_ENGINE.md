## 1. Proposito

Definir el motor conceptual de Inspeccion Tecnica Inteligente.

## 2. Principio ITI

Toda actuacion genera una ITI. Reparacion, mantenimiento e instalacion finalizan con evaluacion tecnica.

La ITI debe capturar estado tecnico, seguridad, operatividad, criticidad, fotos, observaciones, deficiencias, recomendaciones y posibles oportunidades comerciales.

## 3. Informes inteligentes

Una ITI puede generar:

1. Informe tecnico.
2. Informe de deficiencias.
3. Informe de riesgos.
4. Oportunidades comerciales.
5. Recomendaciones tecnicas.
6. Presupuestos derivados.

## 4. Deficiencias

Cada deficiencia debe disponer de estado, prioridad, seguridad, operatividad, criticidad, fotografias, recomendacion, presupuesto, estado comercial e historial.

No desaparece hasta ser resuelta o rechazada.
