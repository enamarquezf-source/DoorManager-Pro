## 1. Proposito

La Knowledge Base centraliza la taxonomia tecnica y el conocimiento validado de DoorManager Pro.

Su objetivo es ordenar familias, fabricantes, componentes, automatismos, sistemas de seguridad, activacion, compatibilidades, fallos, ITI y Gemelo Digital.

## 2. Taxonomia jerarquica

```text
Familia
  Tipo
    Marca
      Modelo
        Configuracion
          Componentes
            Accesorios
              Checklist
                Conocimiento tecnico
```

Nunca se asocia directamente un checklist a una puerta. El checklist pertenece a la jerarquia tecnica.
